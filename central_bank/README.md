# Banking Service

## Description

A vulnerable banking service that allow attacker to steal user's session through cookie.

## 🚀 Features

- Login, signup, logout
- Session management

## 🛠️ Tech Stack

- Node.js
- Express
- MongoDB

## 📦 Installation

```bash
pnpm install
```

## 🚀 Usage

1. Create .env file

2. Run the app

```bash
pnpm start
```

## 📑 API Endpoints

| Endpoint            | Method | Description |
| ------------------- | ------ | ----------- |
| /api/v1/auth/login  | POST   | Login user  |
| /api/v1/auth/signup | POST   | Signup user |
| /api/v1/auth/logout | POST   | Logout user |
