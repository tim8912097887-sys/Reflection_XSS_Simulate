# Malicious Server

## Description

A malicious server that logs stolen sessions from query parameters.

## 🚀 Features

- Log stolen sessions

## 🛠️ Tech Stack

- Node.js
- Express

## 📦 Installation

```bash
pnpm install
```

## 🚀 Usage

1. Create .env file

2. Run the app

```bash
pnpm dev
```

## 📑 API Endpoints

| Endpoint | Method | Description          |
| -------- | ------ | -------------------- |
| /log     | GET    | Log stolen session   |
